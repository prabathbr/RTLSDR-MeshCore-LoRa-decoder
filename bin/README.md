# Binaries
Place built/downloaded files here before running:

| File | Source |
|------|--------|
| `lorarx.exe` | Build with `invoke_build.cmd` (VS 2019 x64) |
| `librtlsdr.dll` | [rtlsdr-bin-w64_static.zip](https://github.com/librtlsdr/librtlsdr/releases/download/v0.9.0/rtlsdr-bin-w64_static.zip) |

`activate_meshcore.bat` and `meshcore.bat` add this folder to `PATH`.
